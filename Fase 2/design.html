Sitio web que brinda aprendizaje sobre los Flex Box y CSS Grid

En este sitio se quiere brindar información sobre la las ventajas
de Flex Box y CSS Grid en el desarrollo de sitios web

Primer borrador

El Flex Box: es un sistema de elementos flexibles que llega con una mecánica mas
potente, limpia y personalizable, en la que los elementos html se adaptan
y colocan automáticamente y es mucho mas facil personalizar los diseños.

Manz. (s.f.). Flexbox CSS. Obtenido de Lenguajecss:
https://lenguajecss.com/css/maquetacion-y-colocacion/flexbox/

CSS Grid: es un sistema de filas y columnas en dos dimensiones y tienen total
libertad de organizar los ítems y el contenido a gusto.

Jerez, N. (28 de Mayo de 2019). Css Grid, la mejor opción para crear diseños web.
Obtenido de medium:
https://medium.com/sue%C3%B1os-graficos/css-grid-la-mejor-opci%C3%B3n-para-crear-dise%C3%B1os-web-b1b7b8735566
